# Assets Directory

Place your app assets here:

## Required Files

- `icon.png` - Your app icon (used in feature graphic)
- `mascot.png` - Optional mascot/character image

## Screenshots

Place device screenshots in `screenshots/`:

- `screenshots/home.png` - Your app's home screen
- `screenshots/feature-1.png` - Feature demonstration
- `screenshots/feature-2.png` - Another feature
- etc.

## Tips

1. **Screenshot dimensions** should match your device resolution
2. **PNG format** recommended for best quality
3. **Transparent backgrounds** for mascots work best
4. Use **consistent naming** across languages (e.g., `home.png` for all)
